# Logo & Branding Images

Letakkan logo dan gambar branding di folder ini.

## Gambar yang Bisa Ditambahkan

1. **logo.png** - Logo utama (transparan)
2. **logo-white.png** - Logo putih untuk dark background
3. **favicon.ico** - Favicon website

## Ukuran yang Disarankan

| File | Ukuran |
|------|--------|
| Logo | 200x200 px |
| Favicon | 32x32 px atau 64x64 px |
