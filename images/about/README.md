## Panduan Pengambilan Foto

- Foto tim teknisi dalam seragam (jika ada)
- Foto saat teknisi bekerja dengan rapi
- Foto peralatan service yang lengkap
- Ukuran minimal: 600x400 px
- Ratio: 3:2

- Ganti dengan foto asli untuk tampilan lebih profesional
