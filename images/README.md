# Images Folder Structure

Folder ini berisi semua gambar yang digunakan di website Service AC Kota Wisata.

## Struktur Folder

```
images/
├── hero/          # Gambar untuk Hero Section
├── gallery/       # Foto dokumentasi pekerjaan teknisi
├── services/      # Icon atau gambar layanan
├── testimonials/  # Foto pelanggan (opsional)
└── logo/          # Logo dan branding
```

## Panduan Penamaan File

- Gunakan huruf kecil dan tanda hubung (-)
- Contoh: `service-ac-1.jpg`, `cuci-ac-outdoor.jpg`
- Format yang didukung: JPG, PNG, WebP

## Ukuran Gambar yang Disarankan

| Section | Ukuran | Ratio |
|---------|--------|-------|
| Hero | 800x600 px | 4:3 |
| Gallery | 600x450 px | 4:3 |
| Logo | 200x200 px | 1:1 |
